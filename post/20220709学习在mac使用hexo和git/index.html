20220709学习在mac使用hexo和git

2022-07-10 2 min read # git # BLOG # 网络 # apple

为了再MAC M1上实施vs code的hexo，绞尽了脑汁。

  1. /bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
    这个是安装brew，也不知道这些mac os是为了什么，加上国内的网络墙，这么个东西也在墙的范围内，这不是技术嘛。搞了一个多小时，加上不好评价的vim。最终找到这个国内的源，才算基本解决。
  2. 还有就是python，我看在mac上我是不想用python了，原来的程序不知道怎么改了才能在mac上使用，这些都涉及到操作系统了。还是就用mac来看看爱奇艺、打打字写写文章吧。
  3. 重新安装hexo和git，还有就是SSH的github，对我这个非计算机科班出身的中年人来说，还是有难度的，尤其是全部一个人在网络上照搬照抄。
  4. 今天决定，在mac上就是写文字和看电影电视剧了，不搞其他的，太心累了。
  5. 把win的联想电脑放回家中使用，在办公室就用这个mac了。
