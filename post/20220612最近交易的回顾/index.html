20220612最近交易的回顾

2022-06-12 7 min read # 交易 # 行为 # 心理 # 记录

最近准备将自己的交易记录和心路历程写出来，然后一年后再看看，分析一下。究竟有哪些做对了，那些做错了。

总体持仓情况

目前自己持有的是70W的项目资金。配置的有外地的基金，然后其余的配置国内股市公司和债券基金。具体配置如下：

目前持有的股票有：三全、红旗连锁、凌霄泵业、中颖电子、伊之密、欧普康视、梅花生物、明阳智能、中国中免、新洁能、卓越新能。下面我先分别描述下为什么这样持仓，基本考虑是什么，下一步如何就大类资产的分布的考虑。再就是对持有的股票、基金的分析，为什么买，以及下一步准备如何处理，包括持有时长、卖出价格、基本面考虑等事项做个记录和计划。

大类配置

我这次大类配置主要是外股基金和大A的股票。配置外股基金主要是考虑美元升值和资产平衡计划。不配置债券，是因为自己这两年需要有比较好的收益用于工作、生活，大头放在在A股，希望能有比较好的收益。大类资产配置基本不再改变。购买的债券基金准备慢慢减仓，每个月按3——5的比例来，再合适的阶段全部转换为A股。

外股基金配置

外股基金现阶段配置了两支，分别是上投摩根全球天然资源大约7.5，南方美国REITs精选LOF6,一方面能源估计未来3年都是涨价的，另一方面REITs能获得固定收益。但是没想到买入的一个月，能源是涨了0.83，但是REITs却跌了1.3，我没有搞明白是怎么回事。

问题分析

  • 美国通胀5月份已经达到了8.6，接近30年来的最高值，美联储连续加息，按道理REITs作为类债券类的资产，肯定价格要下跌，地产价格下跌。但是我感觉REITs不应该啊。收益固定，不至于跌了这么多，累计近14%了。通胀情况下房地产价格不涨吗？看来需要好好研究下美国REITs的基本情况。
  • 关于全球资源，预计是因为石油等大宗商品的涨价，这类基金这一段时间都在大涨。我其实是看好这块资源的长期优势，预计在未来2年都还是会走上坡路吧。

未来策略

短期，还是持有不动。两个基金对冲下来只有约0.5的亏损，应该说配置还不错。下来要好好研究一下REITs的性质，按道理自己买入的时候是考虑得波动较小的品种，怎么波动这么大，比大盘波动都还大。如果可行应该在下个月美联储加息后卖出。至于全球资源基金，应该还可以持有，毕竟大宗商品已经走了近10年的熊市，现在才刚刚抬头，还可以有一个50%——100%的机会。

A股分析

截止6月10日，持有11只股票，分别是三全食品、红旗连锁、凌霄泵业、中颖电子、伊之密、欧普康视、梅花生物、明阳智能、中国中免、新洁能、卓越新能。最多的是欧普康视，大约15%左右。最少的是红旗连锁，大约6%。我先说的是这几个公司买入的理由:

  1. 都是ROE_dt在15%以上，最好的可以到25%。这是最基本的要求（除了红旗连锁，因为买入时观察了大半年，股价也有异动，希望能博一个短差）。
  2. 行业比较分散。有半导体芯片、消费行业、制造业、新能源、食品
  3. 股价基本都在低位。目前看PE都比较低，梅花生物在9倍左右，最高的欧普康视60倍，但是股价基本是最低，已经打了3折了。
  4. 趋势上看，基本都是5日均线上传20日均线。走势都比较健康。

未来A股策略

  • 考虑将上述公司的招股说明书和2021年年报和2022年一季报全部下载下来看看。不一定全部通读，但是必须要好好看看，心里有个底。比如读了中颖电子、凌霄泵业、梅花生物、伊之密，感觉就比较好。至少这几个公司是在做事情，管理层也是比较有思想的，战略比较明确。当然不他们一定完全明了战略的深层次是否符合外部条件和自身资源能力。但是对外部市场和竞争对手的分析，对自身产品、业务的规划，还是比较到位的。
  • 择机卖出红旗连锁，至少在ROE上不达标。
  • 是否卖出中颖电子，还需要再考虑。因为最近都在说芯片产能过剩的问题，但是中颖一直比较稳健，市场也不交有竞争力，在电源管理芯片有较好的市占率。是不是再考虑一年后再说。
  • 还是保持6——10这样的数量，因为这样的数量自己心里不慌张。

总结

  • 左侧下跌时买入。最近因为A股反弹，从2800反弹到3200，我持有的股票基本都已经翻红，有的涨的也很不错，包括最近买入的卓越新能，梅花生物等都是在低位买入，不过两天就上涨，一举跨过成本线。一方面是股票选的好，另一方面是买点也选的比较好。总结起来在牛市或者反弹期初，自己可以在左侧入手。这样心理不慌张，不能买在高位，第一天上涨时买入，当天或者第二天就被套住，心里会很不舒服。这样持股的感觉会很不好。
  • 均线金叉后的下跌时买入。这体现在欧普和新洁能。没一个股票下跌后，一般不会V型反转，都是慢慢起来然后盘整一段时间，可以在感觉盘整到位的时候，均线金叉且贴近长均线的时候买入，感觉也比较好。
  • 总结了两点买入，但是卖出时间还没有选择好。毕竟不卖出，后面又都回去的次数也不少。我希望是长期持股，但是谁知道中国的上市公司后面会出什么问题呢。所以还是每个季度评判下，如果下跌超过正常的比如比前期最高价下跌了20%左右，或者均线死叉，且大盘走势不好。
