PYTHON最大的作用就是用很多有用的程序包，很多功能直接拿来就可以用。用motplotlib,很轻松就可以画出很多我们常见的图标，什么散点图、折线图、条状图等等。

```
import matplotlib.pyplot as plt
import numpy as np

# 多个figure
x = np.linspace(-1, 1, 50)
y1 = 2*x + 1
y2 = 2**x + 1

# 使用figure()函数重新申请一个figure对象
# 注意，每次调用figure的时候都会重新申请一个figure对象
plt.figure()
# 第一个是横坐标的值，第二个是纵坐标的值
plt.plot(x, y1)

# 第一个参数表示的是编号，第二个表示的是图表的长宽
plt.figure(num = 3, figsize=(8, 5))
# 当我们需要在画板中绘制两条线的时候，可以使用下面的方法：
plt.plot(x, y2)
plt.plot(x, y1, 
         color='red',   # 线颜色
         linewidth=1.0,  # 线宽 
         linestyle='--'  # 线样式
        )

plt.show()
```

这段代码就是画两条曲线。很神奇。不过没有概率论知识，我们又怎么用呢?这段时间来学习下概率论和统计基础吧。

了解下，概率为什么对我们很重要。
在1940年，南非金山大学的数学家约翰.克里奇到了丹麦，很不幸正好被德国军队搜捕到投入了丹麦的监狱。在集中营他和狱友没事做了一个概率方面的小实验：扔硬币，并记录每次扔下来的结果。大致结果如下：
总的次数|正面|反面|正反差
:-|-|-|-
100|44|56|-12
2000|1013|987|26
4000|2029|1971|58
10000|5067|4933|134

你看，随着次数的增多，正面向上的结果反而越来越大。
平均律可以说是一条宇宙定律，是经过严格的数学推理得出的。我们在工作、生活中经常都用这条来指导工作生活的。比如：我们投资的股票亏损了，我们常常安慰自己，他还会回到平均线上的。可实际上它真的能再回到均值吗？
是我们常说的平均律出现了问题了吗？还是说“各种结果趋于平均”究竟意味着什么我们还没有真正理解？
