2022-07-20 3 min read # 投资 # 生活 # 原则

上次，我们聊到了风险思维对人生的影响，考虑任何问题，都必须考虑到选择后的风险问题。俗话说得好，天下没有白吃的午餐，天上也不会掉馅饼下来。
今天我们聊聊舍得思维对人生选择的影响。舍得就是常说的要不吝啬今天辛苦换来成果。投资思维为什么重要，就是因为要让人明白今天的辛苦，可以换来明天的成功。如果没有今天现在的舍得，就不会有后天的成功和喜悦。
